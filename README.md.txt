# Accident Detection and Severity Prediction

This project uses Machine Learning to predict the severity of road accidents based on factors such as weather, road conditions, and vehicle details.  
It provides an interactive dashboard built with Streamlit for users to input accident parameters and get predictions.

## 🚀 Features
- Data preprocessing and model training
- Random Forest model for severity prediction
- Streamlit web app for live predictions
- Visualizations and sample dataset

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
